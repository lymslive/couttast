◆ TIME_TAST

#define TIME_TAST (   ...)    G_TASTMGR->TimeTast(__VA_ARGS__)

Run named tast or void function several times, return the average elapse microsecond.

1-3 arguemnt: TIME_TAST(name, times=10, msleep=1000) The 3rd argument means sleep several milli-second after each run and print some verbose message, pass 0 to suppress this behavior. In the target tast or void function, may use TIME_TIC or(and) TIM_TOC to reduce the measured time range.

Detailed Description

Support custome test suite class to group relative test cases.

Author
lymslive
Date
2024-04-10

Each test case in the suite class, defined by DEC_TAST(suite, ..) implies a class derived from suite, and can use the same data member and method, but within separate fresh instance so not affect each other. However, there can be also static data and methods that do need share in more test case.
